- **문제 번호**: #1
- **링크**: [문제 링크](https://leetcode.com/problems/two-sum/)
- **난이도**: 🟢 쉬움
- **유형**: 자료구조 - 해시
- **설명**: 배열에서 두 수를 더해 목표값을 만드는 인덱스를 찾는 문제
